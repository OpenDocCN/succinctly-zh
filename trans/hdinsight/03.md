# 第三章使用 Azure 上的 HDInsight 平台进行简单的情感分析

在本书中，我们将讨论如何在 Windows Azure 上使用 HDInsight 平台执行简单的、基于单词的情感分析练习。该过程将包括几个步骤:

*   创建和配置 HDInsight 集群
*   将数据上传到 Azure Blob 存储
*   使用 C#流创建映射器来分解消息中的单个单词
*   将该映射器作为 Hadoop 映射减少作业执行
*   使用猪来:

*   将情感指示器应用于消息中的每个单词
*   在信息和文字中聚合情感
*   将聚合结果导出回 Azure Blob 存储

*   使用 Hive 向 ODBC 公开结果
*   使用 PowerPivot 添加上下文
*   使用 PowerView 可视化
