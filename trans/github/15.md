除了使用 Markdown，您还可以在 GitHub 文本框中添加表情符号。表情符号是在文档正文中显示的迷你图片，而不是文本。表情符号名称由冒号分隔。如果您在文本窗口中键入一个冒号，GitHub 将显示可用的表情符号，以冒号后的字母开头:
