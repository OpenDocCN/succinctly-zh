# 前言

## 简介

Angular.js 是谷歌开发的开源 JavaScript 框架。它为 JavaScript 开发人员提供了一种高度结构化的方法来开发丰富的、基于浏览器的应用程序，从而提高工作效率。

如果你正在使用 Angular.js 或者正在考虑使用它，这本烹饪书为你可能面临的问题提供了易于遵循的食谱。每个食谱解决一个特定的问题，并提供一个解决方案和对它的深入讨论。
